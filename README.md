# Electric Motor Temperature Prediction

This project predicts electric motor temperature using Machine Learning.

## Tools Used
- Google Colab
- Python
- Scikit-learn
- Flask
- VS Code

## Project Flow
Data → Preprocessing → Model Training → Flask App → Prediction
